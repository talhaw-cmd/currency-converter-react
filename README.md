# 💱 NEOEXCHANGE – Currency Exchange App

## 📌 Overview

NEOEXCHANGE is a simple and user-friendly currency converter built using React.js. It allows users to convert amounts between different currencies instantly using predefined exchange rates.

This project focuses on clean UI design and real-time interaction without using any external API.

---

## 🚀 Features

* Enter any amount to convert
* Select currencies (From → To)
* Instant conversion calculation
* Country flags display
* Currency symbols support
* Fast and responsive UI

---

## 🛠️ Technologies Used

* React.js (Functional Components & Hooks)
* JavaScript (ES6)
* Tailwind CSS
* Static Data (Local currency rates)

## ⚙️ How It Works

1. User enters an amount
2. Selects From and To currency
3. App finds both currencies from data
4. Applies conversion formula:

Converted Amount = (Input / From Rate) * To Rate

5. Result updates instantly

## ▶️ Getting Started

1. Clone the repository
   git clone [https://github.com/your-username/neoexchange.git](https://github.com/your-username/neoexchange.git)

2. Navigate to project folder
   cd neoexchange

3. Install dependencies
   npm install

4. Run the app
   npm run dev
